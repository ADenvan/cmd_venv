# Rules / Skills / Hooks / Subagents

- Prompts -> Создать шаблон для чата.
- Commands -> Быстрые повторяющеися действия.
    - Задать задачу в виде команды. Чтобы не описывать каждый раз одну и туже задачу.
- Subagents -> Изолированны AI assistants. Создать подчиненного агента. Чтобы он выполнял свою задачу и возвращал результат.
- Rules -> Контекст и инструкция.
    - Задать правила для выполнения задач. Например: Не выдумывать код если не знаешь как его написать, не писать лишний код, не усложнять код, не писать лишние комментарии и т.д.
- Skills -> Специализированные workflow + скрипты.
    - Задать скрипты последовательнои исполнения задач. Например: Сначала выполнить задачу 1, потом задачу 2 и т.д.
    - Задать навыки для выполнения задач. Например: Языки программирования, фреймворки, библиотеки, инструменты и т.д.
- Hooks -> Автоматизация и контроль событий.
    - Перед кои-то событием или после события делать какую-то проверку, Запустить помет или скрипт в определенный момент времени..

# Hooks -> запускаются в определенные моменты.
https://cursor.com/docs/hooks#quickstart

1. Хуки работают как с Cursor Agent (Cmd+K/Agent Chat), так и с Cursor Tab (встроенное автодополнение), но используют разные события хуков:
![альтернативный текст](images\Hooks-Документация.jpg)


## 2. Быстрый старт
- Создайте hooks.jsonфайл. Вы можете создать его на уровне проекта ( <project>/.cursor/hooks.json) или в своей домашней директории ( ~/.cursor/hooks.json). Хуки уровня проекта применяются только к конкретному проекту, в то время как хуки домашней директории применяются глобально.
![альтернативный текст](images\Hooks-Создание-Быстрый-старт.jpg)


## 3. Тпы крюков
- Хуки поддерживают два типа выполнения: на основе команд (по умолчанию) и на основе приглашения командной строки (оценивается с помощью LLM).
- Перехватчики команд
``` json
{
  "hooks": {
    "beforeShellExecution": [
      {
        "command": "./scripts/approve-network.sh",
        "timeout": 30,
        "matcher": "curl|wget|nc"
      }
    ]
  }
}
```
- Хуки на основе подсказок
``` json
{
  "hooks": {
    "beforeShellExecution": [
      {
        "type": "prompt",
        "prompt": "Does this command look safe to execute? Only allow read-only operations.",
        "timeout": 10
      }
    ]
  }
}
```
